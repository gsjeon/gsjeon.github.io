K8S Node health check
=====================

Kubernetes Cluster 에서 Node 의 Health 체크를 어떻게 하는지 알아보자.


내가 궁금한건 Node 가 장애가 생겨 강제로 Shutdown시 Kubernetes 내부적으로 어떻게
Health 체크를 하고, Failover 가 이루어지는 궁금하다.


> k8s version 1.16 을 사용합니다.


### 분석

```bash
[clex@taco2-adm-001 ~]$ kubectl get po -n openstack |grep neu
```
